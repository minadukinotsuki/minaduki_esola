# 水無月絵空 X運用OS

XアナリティクスCSVを読み込み、投稿タイプ別に勝ち筋を見える化するStreamlitダッシュボードです。

## 何ができるか

- XアナリティクスCSVを複数アップロード
- 投稿別に以下を集計
  - インプレッション
  - エンゲージメント
  - いいね
  - 返信
  - リポスト
  - ブックマーク
  - 詳細クリック
  - プロフィールアクセス
  - 新しいフォロー
- 投稿タイプを自動タグ付け
  - 体型比較
  - 価格ツッコミ
  - 女性誌レビュー風
  - ストッキング図鑑
  - 国別比較
  - スナップ写真
  - SNS考察
- タグ別の平均値を比較
- 勝ち投稿TOP20を表示
- 次に作る投稿テーマを提案

## セットアップ

```bash
python -m venv .venv
source .venv/bin/activate  # Windowsは .venv\Scripts\activate
pip install -r requirements.txt
streamlit run app.py
```

## 使い方

1. XのアナリティクスCSVをエクスポート
2. アプリ左側からCSVをアップロード
3. 自動タグを確認
4. 必要なら manual_tag を修正
5. タグ別にどの型が強いか確認
6. 次投稿案を作る

## 指標の見方

水無月絵空では、単純なインプレッションだけでなく、以下を重視します。

- プロフィールアクセス率：フォロー・note・ブランド化に近い
- 返信率：会話型バズの強さ
- 詳細クリック率：画像内情報・違和感・確認欲の強さ
- リポスト率：外部拡散
- ブックマーク率：保存用図鑑としての価値

# Codexへの依頼文

このリポジトリは、水無月絵空（@minaduki_esola）のX投稿分析ダッシュボードです。
以下の改善をしてください。

## 追加してほしい機能

1. 手動タグをCSVとして保存・再読み込みできる機能
2. 投稿本文から「問いかけ型」「価格ツッコミ型」「比較型」「図鑑型」を自動判定する機能
3. タグ別に以下をランキング表示
   - インプレッション効率
   - プロフィールアクセス率
   - 返信率
   - 詳細クリック率
   - フォロー率
4. 勝ち投稿を選ぶと、次の投稿案を3パターン出す機能
5. Streamlit UIをもっと見やすくする

## 注意点

- 日本語UIを維持する
- まず動くことを優先
- 大きな外部依存は増やさない
- データはCSVベースで扱う
- テストを追加する

# AGENTS.md

## 目的

このリポジトリは、水無月絵空（@minaduki_esola）のX投稿分析・投稿設計支援ツールです。

## 開発ルール

- UIは日本語。
- まずローカルStreamlitで動くことを優先。
- 複雑なDBは使わない。CSVアップロードで完結させる。
- 投稿タイプ分類は最初はルールベースでよい。
- 指標はインプレッション偏重にしない。
- プロフィールアクセス率、返信率、詳細クリック率を重視。
- センシティブ判定に寄りやすい投稿型は「価格」「女性誌レビュー」「商品ページ風」「診断」に寄せる提案を出す。

## 重要な思想

X投稿は「いい画像」ではなく「どの行動を起こさせる画像か」で評価する。

重視行動：
- 画像拡大
- 滞在
- 詳細クリック
- 返信
- 引用
- リポスト
- プロフィールアクセス
- フォロー

避ける運用：
- 雑な量産
- 投稿して放置
- 外部リンク多用
- 露骨なベイト
- 同じ型の連投
